I'm going to create a probability space for a greatly simplified version of Ross' Exercise 3.10.

TASK 1: Create a probability space for the following experiment. We have a 2-side die, with values of 1 and 2. We throw this die once and obtain a value \(N\). Then, we throw the die \(N\) more times. We are looking for a probability space for this experiment that fulfills the following conditions:
  • \(N: \Omega \rightarrow \mathbb{R}\) is a random variable that represents the value we obtain the first time we throw the die. Equivalently, it represents the extra number of times we have to roll the die. Takes values on \(\{1, 2\}\).
  • \( X_1: \Omega \rightarrow \mathbb{R}\) is a random variable that represents the value we obtain the second time we roll the die. Takes values on \(\{1, 2\}\).
  • \( X_2: \Omega \rightarrow \mathbb{R}\) is a random variable that represents the value we obtain the third time we roll the die. Takes values on \(\{1, 2\}\).
  • \(P\{N = 1\} = 1/2\), \(P\{N = 2\} = 1/2\).
  • \(P\{X_1 = 1\} = 1/2\), \(P\{X_1 = 2\} = 1/2\).
  • \(P\{X_2 = 1\} = 1/2\), \(P\{X_2 = 2\} = 1/2\).
  • \(N\), \(X_1\) and \(X_2\) are pariwise independent.
TASK 2: Calculate \( E\left[ \sum_{i=1}^N X_i \right]\).

Task 1

We define a probability space \((\Omega, \mathcal{F}, P)\) as follows:

  • \(\Omega = \{(n; x_1, x_2 ) \mid n, x_1, x_2 \in \{1, 2\}\}\), that is: \(\Omega = \\ \{(1; 1, 1), (1;, 1, 2), (1;, 2, 1), (1; 2, 2), (2; 1, 1), (2; 1, 2), (2; 2, 1), (2; 2, 2)\}\)
  • \(\mathcal{F} = \mathcal{P}(\Omega)\), the superset of \(\Omega\).
  • \(\Omega\) has the uniform probability: \( \forall A \subseteq \Omega: P(A) = \frac{|A|}{|\Omega|}\frac{|A|}{8}\)

And the random variables: \begin{align} N: &\Omega \rightarrow \mathbb{R}\\ &(n; x_1, x_2) \mapsto n \end{align} \begin{align} X_1: &\Omega \rightarrow \mathbb{R}\\ &(n; x_1, x_2) \mapsto x_1 \end{align} \begin{align} X_2: &\Omega \rightarrow \mathbb{R}\\ &(n; x_1, x_2) \mapsto x_2 \end{align}

The probability us uniform so it's very easy to see that these definitions fulfill the above requirements.

Problems with task 1

The last requirement does not make sense in a real scenario. If true, that would imply for example that \( P\{X_2 = 1 \mid N = 1\} = P\{ X_2 = 1\} = 1/2\). We are saying that the probability that the 2nd throw results in a \(1\), given that we threw the die only once, is \(1/2\). This obviously does not make sense. We would like the probability of the 2nd throw being any value, given that we threw the die only once, to be \(0\).

Therefore, to represent reality, we would like to remove the following requirement:

  • \(N\), \(X_1\) and \(X_2\) are pariwise independent.

And add the following ones:

  • \(P\{ X_1 = 1 \mid N = 1 \} = P\{ X_1 = 2 \mid N = 1 \} = 1/2 \), and \(P\{ X_1 = 1 \mid N = 2 \} = P\{ X_1 = 1 \mid N = 2 \} = 1/2\).
  • \(P\{ X_2 = 1 \mid N = 1 \} = P\{ X_2 = 2 \mid N = 1 \} = 0\), and \(P\{ X_2 = 1 \mid N = 2 \} = P\{ X_2 = 1 \mid N = 2 \} = 1/2\).
